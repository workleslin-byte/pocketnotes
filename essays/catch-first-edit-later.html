Method · 4 min read · April 2026

Catch First.
Edit Later.

The first line is rarely the right one. It only matters that it's caught. Editing without a draft is editing nothing.

Somewhere between the shower and the kitchen, the idea dissolved. You had it. Something about the project, or the conversation, or the thing you'd been trying to articulate for three weeks — it was there, fully formed, and then it wasn't. You can remember that you had it. You cannot remember what it was.

This is not a memory problem. This is a capture problem. The thought existed for about ninety seconds in a form that could have been written down, and then the next thing arrived and displaced it. This is how thought works. It does not queue politely. It does not wait.

The instinct most people have in this situation is to wait for the better version. To hold off writing until the thought is ready — clearer, more complete, worth committing to a page. This is the wrong instinct. The thought you are waiting to improve is the thought you already lost. What arrives next is not the refined version. It is a different thought entirely, slightly guilty about the one that preceded it.

The catch is the thing. The first draft is not the work — it is the permission to do the work. Hemingway, who had strong opinions about most things, had a particularly useful one about first drafts that has survived in various forms across a century of misquotation: the first draft of anything, he maintained, is bad. This was not pessimism. It was liberation. If the first draft is supposed to be bad, then you can write it immediately, freely, without the paralysis that comes from wanting it to be good before it exists.

"The first draft of anything is shit."

Ernest Hemingway, attributed — first recorded in Larry W. Phillips, ed., Ernest Hemingway on Writing, 1984

The notebook's job in this system is simple: it is where the catch happens. Not where the thought becomes good — that happens later, elsewhere, after you've slept and read and come back to it. The notebook is where the thought survives the first forty-eight hours, which is the only window it has.

Write it down badly. Write it down incompletely. Write the half-sentence, the fragment, the thing that doesn't quite make sense yet. The edit is a separate act, and it can only happen if there is something to edit. Editing nothing produces nothing. Editing a bad first draft produces something that might become something else.

Catch first. The refining can wait. The idea cannot.
